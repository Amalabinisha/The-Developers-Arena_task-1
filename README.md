# The-Developers-Arena_task-1
# Personal Introduction Program

## Project Overview

The Personal Introduction Program is a simple Python application that collects basic information from a user and displays a personalized profile and welcome message.

This project was created to practice fundamental Python concepts such as:

* User Input
* Variables
* String Formatting
* Output Display
* Basic Program Structure

## Project Objectives

* Collect information from the user using `input()`
* Store user data in variables
* Display a formatted personal profile
* Generate a personalized summary message
* Build confidence in writing interactive Python programs

---

## Features

* Takes multiple user inputs
* Creates a personal profile
* Displays a personalized summary
* Uses Python f-strings for formatting
* Beginner-friendly and easy to understand

---

## Technologies Used

* Python 3.x

No external libraries are required.

## Setup Instructions

Install Python 3.x on your system.

2. Open the project folder.
3. Open Terminal or Command Prompt.
4. Run the following command:

```bash id="z4vl96"

```

## Code Structure

The program follows these steps:

1. Display the program title.
2. Collect user information.
